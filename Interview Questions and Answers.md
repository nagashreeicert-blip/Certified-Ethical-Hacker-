1. What is Ethical Hacking?

2. What does a CEH do?

Answer:
A CEH performs penetration testing, identifies security gaps, reports vulnerabilities, and recommends fixes to protect systems from malicious attacks.

3. What is the difference between a hacker and an ethical hacker?

Answer:
A hacker exploits systems without permission, whereas an ethical hacker has legal authorization to test and secure systems.

4. What is Footprinting?

Answer:
Footprinting is the process of gathering information about a target system before launching an attack (e.g., WHOIS, DNS lookup, Google hacking).

5. What is Scanning?

Answer:
Scanning identifies live hosts, open ports, vulnerabilities, and network services using tools like Nmap, Nessus, or OpenVAS.

6. What is a Vulnerability?

Answer:
A weakness or flaw in a system that can be exploited by an attacker.

7. What are the types of hackers?

Answer:
White Hat (ethical), Black Hat (malicious), Grey Hat (mixed), Script Kiddies, Nation-State Actors, Hacktivists.

8. What is Social Engineering?

Answer:
A manipulation technique used to trick individuals into revealing sensitive information or granting unauthorized access.

9. What is a Firewall?

Answer:
A security system that monitors and controls incoming/outgoing traffic based on predefined rules.

10. What is Malware?

Answer:
Malicious software designed to harm, exploit, or disrupt systems (e.g., viruses, worms, Trojans, ransomware).

11. What is Enumeration?

Answer:
The process of extracting information such as usernames, shares, network resources, and system details after scanning.

12. Difference between Vulnerability Assessment and Penetration Testing?

Answer:

Vulnerability Assessment: Finds and lists vulnerabilities

Penetration Testing: Attempts to exploit vulnerabilities

13. What are common CEH tools?

Answer:
Nmap, Metasploit, Burp Suite, Wireshark, Nikto, Aircrack-ng, John the Ripper.

14. Explain SQL Injection.

Answer:
An attack where malicious SQL queries are inserted into input fields to manipulate or access the backend database.

15. What is a Man-in-the-Middle (MitM) attack?

Answer:
An attack where an attacker intercepts communication between two parties.

16. What is Password Cracking?

Answer:
Recovering passwords using brute force, dictionary, rainbow tables, or hybrid attacks.

17. What is a Zero-Day Exploit?

Answer:
A vulnerability that is exploited before the vendor becomes aware and releases a patch.

18. What is OSI Layer significance in hacking?

Answer:
It helps ethical hackers understand how attacks occur at each layer (e.g., sniffing in Layer 2, DoS in Layer 3).

19. What is IDS vs IPS?

Answer:

IDS: Detects attacks

IPS: Detects and blocks attacks

20. What is Web Application Penetration Testing?

Answer:
Testing web apps for vulnerabilities like XSS, CSRF, SQL injection, authentication flaws.
